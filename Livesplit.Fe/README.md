# LiveSplit.Fe
Autosplitter for the game Fe

## Installing
  * Open LiveSplit
  * Edit your splits
  * Set the game to Fe
  * Click Activate autosplitter
  
## Splits
  * Starts when you are able to move
  * Splits :
    * Unlock a new perk
    * Unlock a new voice
  * Ends when you loose the control of the seeker
