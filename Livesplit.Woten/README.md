# LiveSplit.Woten
Autosplitter for the game Woten

## Installing
  * Open LiveSplit
  * Edit your splits
  * Set the game to Woten
  * Click Activate autosplitter
  
## Splits
  * Starts when on the first map
  * Splits :
    * Dialog with Aedling
    * Complete a map
    * Dialog with Statues
  * Ends when you touch the pedestal
  * Reset in the main menu
