# LiveSplit.GRIS
Autosplitter for the game GRIS

## Installing
  * Open LiveSplit
  * Edit your splits
  * Set the game to GRIS
  * Click Activate autosplitter

## Splits
  * Starts when selecting "New Game"
  * Ends on black screen
  * Splits :
    * Unlock a color
    * Unlock a power
    * Give the fireflies to the constellation
